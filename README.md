# 🎯 Pinball-Bounce 弹珠游戏介绍

> 🎮 **[🔗 在线试玩 →](https://guiii007.github.io/pinball-bounce/plinko.html)**

这是一个支持高度自定义的弹珠游戏，结合了物理碰撞、多地图切换与丰富的奖励机制。以下是游戏的核心功能说明：

---

## 🗺️ 地图系统
游戏内置了**四款**不同视觉和物理碰撞体的地图：三角形、菱形、正方形、蜂巢。

## 🎮 发射机制
- **自由拖拽发射点**：球发射的点位支持**随意拖动**，便于玩家精确瞄准。
- **力度调节**：发射力度根据**鼠标拖拽的幅度（距离）**动态决定，拖动距离越长，发射力度越大。

## 🛠️ 地图挡板选项
为满足不同的游玩需求，地图挡板提供三种模式：
1. **添加挡板**：系统自动布置标准碰撞挡板。
2. **不添加挡板**：纯地图环境，考验玩家的入射角度。
3. **挡板自主绘画**：支持玩家自定义绘制挡板的形状与位置。

## 🔊 音效反馈
游戏拥有差异化的物理音效：
- 球撞击**挡板**时，触发特定的碰撞音效。
- 球撞击**钉阵**（障碍物）时，触发完全不同的环境音效。

## 🏆 计分与奖励机制
- **底部计分区域**：每张地图的最底部均设有专用的分数统计区域。
- **奖励动画**：地图中隐藏有**奖励小区域**。当弹珠成功落入该区域时，会触发**专属的奖励小动画**。
